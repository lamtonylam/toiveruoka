## Unicafe food finder frontend

## Local development
1. Install dependencies
```
npm install
```
2. Start the development server
```
npm start dev
```
3. Create a `.env` file in the root directory and add the following line, replacing `<backend_url>` with the URL of your backend server (e.g., `http://localhost:3000`)
```
REACT_APP_BACKEND_URL=<backend_url>
```
4. Open your browser and navigate to `http://localhost:5173` to view the application.
